# Проект для тестирования sandbox-tests

### Стек

* Язык программирования: **Python**


### Структура проекта

```
 ├── o3                             # Папка с настройкой CI
 ├── clients                        # Папка с базовыми клиентами
 │   └── http                 # Папка с http клиентами
 ├── checkers                       # Папка проверками для конкретных методов
 ├── config                         # Конфигурационные файлы
 ├── helpers                        # Пакет с классами помощьниками и фасадами
 ├── packages                       # Самописные вспомогательные пакет и клиентские библиотеки
 ├── tests                          # Папка для хранения тестов
 │   ├── smoke                      # Чтобы был запуск в пайплайне тесты должны располагаться в папка smoke/functional
 │   └── conftest.py                # conftest с импортированными, сгенерированными фикстурами
 ├── .gitignore                     # Стандартный файл для игнорирования нежелательных файлов для загрузки в репозиторий
 ├── gitlab-ci.yml                  # Файл с подключением проекта к пайплайну для проектов с тестами
 ├── mypy.ini                       # Файл конфигурации линтера mypy
 ├── telegram-notifier-config.ini   # Файл конфигурации телеграм репортера
 ├── swagger-coverage-config-sandbox-tests.json                # Файл конфигурации для снятия покрытия сервиса автотестами
 └── README.md                      # Файл с описанием репозитория для тестов
```
